# Requirements
1. PHP >= 8.2
2. MySQL/MariaDB server
3. mbstring
4. ffmpeg
5. imagemagick; Optional,needed to make animated thumbnails from GIFs.
6. PHP GD
7. PHP PDO

# How to install

1. create a database
2. Access install.php from your web browser
3. After filling in the fields with your details, click on "install"
4. After doing that, if everything went well, you will be redirected to mod.php. There, enter the administrator credentials you created in the previous step, and you're done.
Remember to test it in a local environment.
